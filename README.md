**E-Commerce Sales Data Analysis Project**

I conducted a comprehensive exploratory data analysis (EDA) on an Amazon e-commerce sales dataset, focusing on sales trends, category performance, and regional distribution across India. Through this project, I uncovered key insights into revenue drivers and market concentration using Python, Pandas, and Matplotlib.

Key Features

Data loading and cleaning from Amazon order reports

Visualizations for sales trends over time, category-wise sales, state-wise sales, and top cities

Summary statistics on quantities sold and revenue by category, state, and city

Technologies Used

Python with Pandas for data manipulation

Matplotlib for plotting sales trends, bar charts (category performance, state-wise sales)

Jupyter Notebook for interactive analysis

Detailed Setup Steps

Clone the repository and open e-commerce_proj.ipynb in Jupyter Notebook or JupyterLab

Ensure Python 3.x environment with libraries: pandas, matplotlib (install via pip install pandas matplotlib if needed)

Run cells sequentially: data import shows order details (Order ID, Date, Status, Amount); EDA generates plots like sales trends and category bars

Review markdown sections for insights ("Set" category leads revenue)

Key Findings
"Set" is the top-performing category with the highest revenue share, followed by "Kurta" and "Western Dress"; low performers like "Saree" and "Dupatta" suggest optimization opportunities. Maharashtra leads state-wise sales, followed by Karnataka and Telangana, indicating regional concentration. Bengaluru tops city sales, highlighting urban market dominance.

**Conclusion**

This analysis highlights sales concentration in specific categories (Sets) and regions (Maharashtra), indicating growth potential through diversification and market expansion. The project demonstrates my practical data analytics skills for e-commerce optimization.
